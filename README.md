# My Website

Astro-powered personal blog focused on Math, Philosophy, and Code.

## Prerequisites

- Node.js 20+
- npm (bundled with Node)

## Development

```bash
npm install
npm run dev
```

## Build & Preview

```bash
npm run build
npm run preview
```

## Deploy

```bash
npm run deploy
```

This runs `astro check`, `astro build`, then publishes `dist/` via `gh-pages`.
